# ComelyMD - 极简 Markdown 分享服务

![Version](https://img.shields.io/badge/version-v1.1.0-blue.svg)

一个基于 `Go + SQLite + Pico.css` 构建的轻量级 Markdown 分享工具。专为低配服务器设计，提供跨平台的纯静态无关联容器环境部署能力，并包含完备的隐私密码锁与生命周期阅后即焚清理架构支持。

## ✨ 核心特性

- **跨平台与低资源占用**：借宿 `modernc.org/sqlite` 纯 Go 驱动，完全满足 `CGO_ENABLED=0` 环境架构编译方案，生成的容器镜像极小，绝佳适配单核小内存服务器。
- **环境隔离访问**：无全局文章目录陈列聚合站，所有对外提供链接经由高随机密度的独立 `Base62` 专属哈希流控制分配。
- **按需加锁并辅导组合分享**：支持自主挂载开启 4 位随机访问密码；搭载了方便的一键点击提取口令密码或组合口令连带连接文字发送功能（类似网盘全自动分享链接获取）。
- **生命周期清扫与阅后即焚**：后台附带内存驱动处理的自动回收计时引擎控制器。并含有强硬阻断的拦截机制进行一旦被成功解封宣读过的链接文档的立即单次阅看回收操作。
- **页面优雅交互**：前端表单通过悬浮交互卡片进行视觉极简引导留白；所有内容阅读展现提供并实现了深浅配对且自动顺延系统的显示变装。
- **查阅源文件便捷解构分离提取**：所有的正常页面支持随附侧边三大浮动控制器辅助执行提取操作动作：能够原样提纯复制基础文本段落、剥壳解离提取为 `.md` 本机文本资源以及转化不带页面导航杂色边的 PDF 高清脱出打印版本。

## 🚀 部署指南 (基于 Github Actions CI 构建)

为了解除低配微型 VPS 上现场编译 Go 及其数据库体系容易诱发的卡顿甚至系统 OOM 崩盘的恶劣问题风险，本发行版推荐依托且仅依赖 Github Runner 进行的镜像脱端预制包发行。

在您的终端，只需一键取自镜像仓库官方拉取源头即可完成全系统运行发布：

```bash
git clone https://github.com/Loxonl/comelyMD.git
cd comelyMD

# 全量跳过极其伤耗低端服务器计算资源的编译链系统，全时使用预出包冷启动极速还原
docker-compose pull
docker-compose up -d
```

> **系统更替版及修补常规升级说明**：
> 因为所有编译动作的远挂托管在 GitHub 流水线中进行，只要等待并确定流水绿皮动作打包完备结束后。只需要在这台小机子上再次无负担敲一行提取拉起指令组合即可：`docker-compose pull && docker-compose up -d`。

## 🛠️ 后台主 API 通道入口

如果独立使用脱落页面做微小存储引擎后台中介调用支持的话可以独立请求这个标准接口接入调用服务：

- `POST /api/pages`
  - 传入主体表单请求体 (`FormData`)：
    - `markdown`: (必填) Markdown 源文本结构段落
    - `is_burn`: (可选) 增加安全一次性丢弃逻辑开启参数："true" 
    - `expire_time`: (可选) 给后台打上时间期限自动化自清引擎刻度参数："5m", "1h", "6h", "24h", "7d", "30d"
    - `with_password`: (可选) 是否强制由安全派号下发四维乱号校验分配："true"
  - 同调回调结构举例回传标准响应体：`{"id": "A4xF8t", "url": "https://[当前主站的公网入口]/p/A4xF8t", "pwd": "M2kA"}`

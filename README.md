# ComelyMD - 随行 Markdown 安全即焚大仓

![Version](https://img.shields.io/badge/version-v1.1.0-blue.svg)

一个极简、低资源占用、极致优雅审美且易于在各类甚至微型云服务器上自动化部署分发的 Markdown 分享服务。基于 `Go + SQLite + 定制型 Pico` 构建。在绝对纯净免维护的部署方案基础上，加持了最高级的单项隐私剥离方案和极客级别的多功能剪贴导出品能。

## ✨ 核心特性

- **极致空间留白与精简沉浸排版**：全面颠覆粗糙的系统级表单设计，注入具备拟态和微小呼吸阴影悬浮卡片的输入区域以及随系统响应式交切渲染的黑白 `Highlight.js` 主题控制台引擎。
- **纯 Go 驱动跨节点通病免疫**：依托 `modernc.org/sqlite` 并结合 Github CI 构建全平台编译释放流，实现零系统依赖的强 `CGO_ENABLED=0` 挂载，产生的十几兆镜像是单核廉价小机的梦中神包。
- **无挂起级绝对隔离**：不设文章列表，切断所有全局导航侧露的风险。只利用高度混乱高维的随机 `Base62` 哈希生成分享大门主链。
- **系统挂载级高级暗语锁**：支持随意叠加四位乱序特工门锁，提供直观的点击隐藏复制机制，更有独一无二的**结合源分享链接及配套密码文字的复合云盘级自动拼合入板**能力，让转发彻底零操作！
- **时钟基准拦截解体与焚毁粉碎**：可精准开启控制在自然时长流逝后的回收抛弃倒计时，亦或者直接拥有一旦资源触达目标用户终端单此后立即释放删除抹平的【绝对级无痕阅后即焚引擎】。
- **降维回落剥离引擎（前端悬浮）**：阅读端搭载了一套免后端的绝赞数据还原浮窗：
  - 📋 **纯净原码剥离**：直接从 DOM 根部深处挖掘剥解原封不动的 Markdown 文本强填入系统剪贴板。
  - 💾 **物理沙盒打包重装**：用高阶的客户端重封装将其转录进标准带标签命名的 `.md` 文件提供下载回机！
  - 🖨️ **降噪全清数字压制**：开启特殊级 CSS 切片媒体拦截并呼叫跨端生成器压制出不带一丝网页边框导航痕迹的无损重构版本高清 PDF ！

## 🚀 部署指南（预制镜像方案）

面对目前微弱低端 VPS 进行全盘代码环境交叉编译引起的长时间拥堵、CPU 卡死打爆、以及 OOM 连环断裂。我们全面摒弃了老旧的原生 `docker build` 方案，改以 GitHub 为云服务器打点。

在你的服务器节点获取并布署最新包时，永远只需要一行即可从零下发与更新：

```bash
git clone https://github.com/Loxonl/comelyMD.git
cd comelyMD

# 秒级拉取由 GitHub 提供的大型流水线编织后的终极静态微容器
docker-compose pull
docker-compose up -d
```

> **日常服务持续集成升版（CI/CD）**：
> 此后你在个人开发机中一旦通过 `git push` 推送了后续特性的新代码。你那什么也不用干的部署站仅需再顺手重挂一下：`docker-compose pull && docker-compose up -d`！

## 🛠️ 后端级核心 API 通道控制口

剥离所有多余的前端显示之后，本路由可提供最为清脆的 RESTful 拦截生成响应服务，供更广的跨站接驳：

- `POST /api/pages`
  - 请求层载体 (`FormData`)：
    - `markdown`: (必备) 源段输入内容
    - `is_burn`: (可选项) 是否设定本次链接的触发单拉即被撕毁："true" 
    - `expire_time`: (可选项) 后台清扫定时器生命期限枚举："5m", "1h", "6h", "24h", "7d", "30d"
    - `with_password`: (可选项) 分配随机乱码防爆护盾："true"
  - 同调回调结构范例：`{"id": "s8mPz9", "url": "https://[发布基准主机域]/p/s8mPz9", "pwd": "M2kA"}`
